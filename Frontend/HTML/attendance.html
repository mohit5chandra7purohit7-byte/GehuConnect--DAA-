🔬 DSA Implementation — Merge Sort

1
Input — 6 subjects with raw attendance data loaded as an array
2
Divide — Array is split into halves recursively until single elements remain
3
Conquer — Each pair of halves is merged back in sorted order (by % ascending)
4
Output — Fully sorted list: subjects with lowest attendance appear first
⏱ Time: O(n log n) 💾 Space: O(n) 📐 Stable: Yes 📁 Implemented in: Backend/attendance.cpp
