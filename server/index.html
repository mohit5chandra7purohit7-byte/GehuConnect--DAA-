⚙ What happens when you click Calculate

1
Browser reads your two numbers and selected operation from the form
2
fetch() sends a GET request to the server endpoint /calculate?a=10&b=4&op=add
3
server.js receives the request, does the math, and sends back {"result": 14} as JSON
4
Browser receives the JSON and displays the result on the page
